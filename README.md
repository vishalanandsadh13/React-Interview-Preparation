# React Interview Preparation (Vite + React)

This repo is a **learning + revision workspace** for core React interview concepts.

## What you get
- **Vite + React (JavaScript)** project
- **Functional components only**
- A `src/concepts/` folder where each concept has:
  - `Example.jsx` (working demo)
  - `README.md` (concise explanation + interview questions)

## Run locally
```bash
npm install
npm run dev
```

Then open the printed local URL (usually `http://localhost:5173` or the next available port).
